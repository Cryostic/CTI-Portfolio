```
├── Winnti toolset confirmed: Cobalt Strike, Gh0st RAT, KEYPLUG

JANUARY 17, 2022 : TREND MICRO FOUNDATIONAL EARTH LUSCA REPORT

├── Comprehensive campaign and infrastructure analysis published

│   - Cyberespionage: government, academic, religious, civil society
```

<h3>Geographic Explosion & Linux Capability (2023)</h3>
<p>Defining characteristic: H1 2023 expansion across Central Asia/Balkans/Latin America; SprySOCKS Linux backdoor; 17-country targeting documented</p>

```
H1 2023 : GEOGRAPHIC EXPLOSION
├── Trend Micro: Earth Lusca "extended its operations, targeting countries 
│   around the world during the first half of 2023"
├── New theaters: Central Asia, Balkans, Latin America, Africa
├── Primary targets: government departments focused on:
│   └── Foreign affairs, technology, telecommunications
├── Method: public-facing server N-day exploitation (shift from social engineering)
│   └── Fortinet (CVE-2022-40684, CVE-2022-39952)
│   └── GitLab (CVE-2021-22205)
│   └── Microsoft Exchange (ProxyShell)
│   └── Zimbra (multiple CVEs)
│   └── Telerik UI for ASP.NET AJAX (CVE-2019-18935)
└── Post-compromise: web shells → Cobalt Strike → ShadowPad/Winnti RAT

AUGUST 10, 2023 : REDHOTEL: 17 COUNTRIES IN 3 YEARS (Recorded Future)
├── Recorded Future Insikt Group publishes comprehensive RedHotel analysis
├── 17 countries targeted 2021–2023: Asia, Europe, North America
├── Victim breakdown: majority government organizations:
│   └── Prime ministers' offices, finance ministries, legislative bodies, interior ministries
├── Dual mission explicitly confirmed: "intelligence gathering AND economic espionage"
├── Chengdu, China administrative infrastructure confirmed
├── "Exemplified relentless scope and scale of wider PRC state-sponsored activity"
└── Prolific operational tempo: consistently one of the most active PRC groups

SEPTEMBER 18, 2023 : SPRYSOCKS LINUX BACKDOOR (Trend Micro)
├── FIRST Earth Lusca Linux-specific backdoor documented
├── SprySOCKS: derives from open-source Windows Trochilus RAT
├── C2 protocol mimics RedLeaves Windows trojan (TCP-based)
├── Features: reconnaissance, remote shell, proxy capability, file operations
├── At least two versions identified (1.1 and 1.3.6) active development
├── Hosted on same Cobalt Strike C2 delivery server as Linux Winnti
└── Only Earth Lusca observed using SprySOCKS (exclusive attribution indicator)
```

<h3>Election Lures, I-SOON Exposure & KTLVdoor (2024–2025)</h3>
<p>Defining characteristic: Taiwan election targeting; I-SOON data leak; FishMonger DOJ indictment; KTLVdoor multiplatform tool</p>

```
FEBRUARY 2024 : I-SOON (ANXUN) DATA LEAK

├── Documents include: contracts, tool inventories, target lists,

├── Confirms: FishMonger = I-SOON operational arm
├── Confirms: multiple PRC government agency clients (MPS, MSS, others)
├── Reveals: employee dissatisfaction (low wages), financial pressure
│   → explains financial motivation attacks as supplementary income
└── SIGNIFICANCE: Most detailed involuntary disclosure of PRC 
    contractor operations in CTI history

DECEMBER 2023 - JANUARY 2024 : TAIWAN ELECTION LURE CAMPAIGN (Trend Micro)
├── Campaign active approximately December 22, 2023 to January 2024
├── Lure: Chinese-Taiwanese relations document
├── File timestamp: December 22, 2023 (two days before Taiwan elections)
│   └── Elections held January 13, 2024 (Lai Ching-te elected)
├── Delivery: archive file (ZIP/RAR) → LNK file → staged Cobalt Strike
├── C2 abuse: domain names impersonating cybersecurity companies
│   └── cybereason.xyz (impersonates legitimate firm Cybereason)
├── Cobalt Strike watermark: 100000000 (consistent across campaigns)
└── Attribution: high confidence to Earth Lusca based on tools + TTPs

MARCH 5, 2025 : U.S. DOJ INDICTMENT: I-SOON EMPLOYEES
├── DOJ unseals indictment of I-SOON employees + MPS officers
├── "Hackers-for-hire" responsible for breaching:
│   └── U.S. federal and state agencies (including Dept. of Treasury)

MARCH 20, 2025 : ESET OPERATION FISHMEDLEY REPORT
├── ESET independently confirms FishMonger = I-SOON operational arm

SEPTEMBER 4, 2024 : KTLVDOOR MULTIPLATFORM BACKDOOR (Trend Micro)

│   ├── Impersonates system utilities: sshd, java, sqlite, bash, edr-agent
│   ├── XOR-encrypted + Base64-encoded configuration
│   ├── Custom TLV-like (type-length-value) format "KTLV" marker
│   ├── Capabilities: command execution, file manipulation, system info,
│   │               port scanning, proxy support, upload/download
│   └── 50+ C2 servers discovered ALL hosted on Alibaba in China
├── ANALYTICAL UNCERTAINTY: "it remains unclear if the entire 
│   infrastructure is exclusive to Earth Lusca" scale suggests
│   possible sharing with other Chinese threat actors
└── Most complex tool in Earth Lusca's documented arsenal to date

2025–2026 : CONTINUED OPERATIONS
└── CYFIRMA confirms Earth Lusca active through at least March 2026
    └── Continued targeting across Asia, Europe, Latin America
    └── No major operational disruption following DOJ indictment
    └── Pattern mirrors APT40: indictment = documentation, not deterrence
```

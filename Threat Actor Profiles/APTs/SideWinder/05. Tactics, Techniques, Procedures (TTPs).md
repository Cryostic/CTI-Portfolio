# 05. Tactics, Techniques, Procedures (TTPs)

<h2>ATT&CK Kill Chain Mapping (Unified Kill Chain: IN --> THROUGH --> OUT)</h2>

<h3>IN: Initial Access & Execution</h3>

| Technique | ATT&CK ID | Description |
| --- | --- | --- |
| Spear-Phishing Attachment | T1566.001 | PRIMARY vector; DOCX/XLSX files and ZIP archives with LNK files; hyper-targeted to victim organization and geopolitical context |
| Spear-Phishing Link |T 1566.002 | PDF files (2025 ClickOnce chain) prompting download of fake Adobe Reader |
| Remote Template Injection | T1221 | DOCX fetches RTF template from attacker C2 at open-time; bypasses static analysis of attachment itself |
| Exploit: CVE-2017-11882 | T1203 | MS Office Equation Editor memory corruption; PRIMARY exploit since ~2018; still primary as of 2026 despite being a 2017 patch |
| Exploit: CVE-2017-0199 | T1203 | MS Office/WordPad remote code execution via OLE object; secondary exploit; confirmed active 2025 |
| User Execution - Malicious LNK | T1204.002 | ZIP archives contain Windows LNK (shortcut) files; victim executes LNK thinking it opens document |
| Trusted Application Proxy (ClickOnce) | T1218 | MagTek "ReaderConfiguration.exe" signed with legitimate certificate; masquerades as Adobe Reader update; 2025 chain |
| Geo-fenced Payload Delivery | T1480 | C2 servers return HTTP 404 to non-target IP ranges; payload only served to IPs within target geographies |
| Drive-by Compromise (AntiBot) | T1189 | SideWinder.AntiBot.Script: server-side redirect logic filtering analysis environments |

---

<h3>THROUGH: Persistence, Defense Evasion & Privilege Escalation</h3>

| Technique | ATT&CK ID | Description |
| --- | --- | --- |
| DLL Side-Loading | T1574.002 | CORE evasion; legitimate signed binaries execute malicious DLLs from same directory; Backdoor Loader always side-loaded; ClickOnce chain uses TapiUnattend.exe |
| Fileless / In-Memory Execution | T1620 | StealerBot NEVER written to disk; loaded entirely in memory; defeats file-based AV and forensic disk analysis |
| Obfuscated Files (Encrypted Storage) | T1027 | StealerBot modules stored encrypted; XOR decryption using first 21 bytes of encrypted file as key (documented in Trellix) |
| Anti-Analysis / Environment Check | T1497 | SideWinder.AntiBot.Script filters automated analysis; WarHawk checks Pakistan Standard Time before executing |
| Scheduled Task | T1053.005 | Persistence for loader component; established post-compromise |
| Registry Run Keys | T1547.001 | Secondary persistence mechanism; complements scheduled task |
| Masquerading | T1036 | Malicious components named as legitimate system utilities; ClickOnce impersonates Adobe Reader |
| Server-Based Polymorphism | T1027.002 | Payload mutated server-side before delivery; defeats hash-based detection |
| Signed Binary Proxy Execution | T1218 | Legitimate signed binaries (MagTek, system components) used as launchers for malicious DLLs |
| UAC Bypass | T1548.002 | StealerBot includes privilege escalation plugin for UAC bypass |
| Reflective Code Loading | T1620 | StealerBot Orchestrator loads plugins reflectively in memory; no disk touch |

<h3>OUT: Collection, C2 & Exfiltration</h3>

| Technique | ATT&CK ID | Description |
| --- | --- | --- |
| StealerBot: Keylogging | T1056.001 | Dedicated keylogger plugin; captures all keystrokes on compromised host |
| StealerBot: Screen Capture | T1113 | Screenshot plugin; captures desktop at configured intervals or on command |
| StealerBot: Credential Theft | T1555 | Browser saved password harvesting; Windows Credential Manager access |
| StealerBot: File Collection | T1005 | Targeted document collection: .doc, .xls, .pdf, .ppt, and configured file types |
| StealerBot: Reverse Shell | T1059.001 | Full remote shell capability for operator command execution |
| StealerBot: Additional Payload Delivery | T1105 | Ingress tool transfer; downloads additional modules or tools on operator command |
| StealerBot: Process Discovery | T1057 | Enumerates running processes; reports to Orchestrator for situational awareness |
| StealerBot: System Profiling | T1082 | OS version, hardware info, user identity collected at initial compromise |
| C2 via HTTP/S | T1071.001 | All C2 traffic via HTTP/S to actor-controlled domains; mimics legitimate traffic |
| Encrypted Channel | T1573 | C2 communications encrypted; module data encrypted at rest (XOR keying) |
| Dynamically Generated C2 Paths | T1568 | ClickOnce chain: each victim receives unique, dynamically generated C2 path |

# 05. Tactics, Techniques, Procedures (TTPs)

<h2>ATT&CK Kill Chain Mapping (Unified Kill Chain: IN --> THROUGH --> OUT)</h2>

<h3>IN: Initial Access & Execution</h3>

| Technique | ATT&CK ID | Description |
| --- | --- | --- |
| Spear-Phishing Attachment | T1566.001 | PRIMARY vector; ISO files delivered via targeted emails; lures include military notices, job applications, diplomatic correspondence |
| Spear-Phishing Link | T1566.002 | Supplementary; links to malicious download pages |
| User Execution | T1204 | Victim opens ISO file and executes signed legitimate binary (triggering DLL side-loading) |
| Exploit Public-Facing Application | T1203 | CVE-2017-0199 exploitation (Microsoft Office Remote Code Execution); observed across multiple campaigns |
| Template Injection | T1221 | Malicious macro template fetched from GitHub when victim opens DOC file within ISO |

---

<h3>THROUGH: Persistence, Privilege Escalation & Defense Evasion</h3>

| Technique | ATT&CK ID | Description | 
| --- | --- | --- |
| DLL Side-Loading | T1574.002 | CORE TECHNIQUE; legitimate signed binaries exploited to load malicious DLLs; evades AV detection |
| Event Triggered Execution: Change Default File Association | T1546.001 | RARE TECHNIQUE; malware registers itself as handler for a common file type, ensuring launch when that type is opened; rarely seen in the wild |
| Boot or Logon Autostart Execution | T1547 | TelePowerBot persists via registry and launches at system boot |
| Scheduled Task / Job | T1053 | Secondary persistence mechanism |
| Browser Extensions | T1176 | Used for browser credential persistence |
| Masquerading | T1036.005 | Malicious files named to match legitimate system components |
| Obfuscated Files / Information | T1027 | KamiKakaBot uses open-source .NET obfuscation engine (ConfuserEx-style); updated post-disclosure |
| Trusted Developer Utilities Proxy Execution (MSBuild) | T1127 | MsBuild.exe used as LOLBin to execute KamiKakaBot payload from XML embedded in decoy document |
| Deobfuscate / Decode Files or Information | T1140 | XOR-encrypted sections in decoy documents decrypted at runtime |
| Virtualization/Sandbox Evasion | T1497 | Checks for analysis environment indicators |
| Process Injection | T1055 | Injecting into legitimate processes for evasion |
| Abuse Elevation Control Mechanism: Bypass UAC | T1548.002 | Privilege escalation for persistence installation |
| Windows Management Instrumentation | T1047 | Used in reconnaissance and lateral movement |

---

<h3>OUT: Collection, C2 & Exfiltration</h3>

| Technique | ATT&CK ID | Description | 
| --- | --- | --- |
| Screen Capture | T1113 | Screenshot capability via custom modules |
| Audio Capture | T1123 | Microphone recording via PowerSploit/Get-MicrophoneAudio (only observed public tool) |
| Credentials from Password Stores | T1555 | Cucky and Ctealer extract passwords, cookies, browsing history from 20+ browsers |
| File and Directory Discovery | T1083 | Enumeration of target file systems |
| System Information Discovery | T1082 | OS and hardware reconnaissance |
| Query Registry | T1012 | Registry interrogation |
| Data Encoding | T1132 | Data encoded prior to exfiltration |
| Web Service(Telegram API) | T1102 | PRIMARY C2 CHANNEL - all C2 communication over Telegram bot API; commands sent via actor-controlled Telegram channel |
| Exfiltration Over Webhook(HTTP) | T1041 | webhook.site used for HTTP-based data exfiltration (added 2023) |
| Replication Through Removable Media | T1091 | USB device infection propagates to USB drives connected to compromised hosts |
| Input Capture; Messaging Application Harvesting | T1417 | ZMsg harvests message content from Telegram, Viber, and Zalo desktop apps |

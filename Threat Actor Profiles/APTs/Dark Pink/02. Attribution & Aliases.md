<h2> Attribution Assessment </h2>

| Attribution Claim | Evidence | Counter-Evidence/Caveats |
| --- | --- | --- |
| Southeast Asian State Next | GitHub upload timestamps consistently align with UTC+7 (Indochina Peninsula Standard Time - Vietnam, Thailand, Cambodia, Laos) per Anheng Security analysis | UTC+7 covers multiple nations; could be time zone spoofing |
| Vietnamese Affiliation | Some researchers note behavioral and TTP similarities to OceanLotus; CocCoc browser targeting (Vietnamese-popular browser) in Ctealer/Cucky | No technical artifact definitively links Dark Pink to APT32; could be shared tooling or common training |
| Chinse State Nexus | EclecticIQ observes interest in ASEAN-Europe diplomatic relations consistent with Chinese intelligence tasking; some TTP similarities to known Chinese actors | EclecticIQ explicitly hedges this as a hypothesis, not a conclusion |
| Indian State Nexus | Both groups exploit CVE-2017-0199; SOCRadar notes this overlap | CVE-2017-0199 is widely exploited by many actors; not a reliable distinguishing indicator |

<h2>Working Assessment</h2>
<p>
  Dark Pink most likely orginates from Southeast Asia based on evidences - UTC+7 timezone alignment, targeting patterns, language artifacts. 
  The group's singular focal targeting on ASEAN Government and Military entities is consistent with nation-state espionage mandate. 
  The attribution to specific state still remains <b>Unconfirmed</b>. 

  The group used naming conventions on email addresses for data exfiltration. These gave the Group-IB researchers the basis to name the group "Dark Pink"
